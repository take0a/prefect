Prefect は Python パッケージとして公開されており、Python 3.9 以降が必要です。Prefect は Python 仮想環境にインストールすることをお勧めします。

<Installation />

Prefect が正常にインストールされたことを確認するには、次のコマンドを実行します:

```bash
prefect version
```

次のような出力が表示されます:

```bash
Version:             3.1.10
API version:         0.8.4
Python version:      3.12.2
Git commit:          d6bdb075
Built:               Thu, Apr 11, 2024 6:58 PM
OS/Arch:             darwin/arm64
Profile:              local
Server type:         ephemeral
Server:
  Database:          sqlite
  SQLite version:    3.45.2
```

<Accordion title="パワーユーザー向けの追加コマンド">


### `uv`を使用する場合

Python 3.12 と `prefect` がインストールされた `ipython` シェルを起動します:

```bash
uvx --python 3.12 --with prefect ipython
```

`uv` 仮想環境にprefectをインストールします:

```bash
uv venv --python 3.12 && source .venv/bin/activate
uv pip install -U prefect
```

アクティブなプロジェクトにprefectを追加します:

```bash
uv add prefect
```

`uvx` を使用して一時的な Python 環境で Prefect Server を実行します。

```bash
uvx prefect server start
```

### `docker` を使用する場合

ローカルマシンの 4200 ポートにポート転送されたコンテナで prefect server を実行します:

```bash
docker run -d -p 4200:4200 prefecthq/prefect:3-latest -- prefect server start --host 0.0.0.0
```

## Windows インストール

Prefect は、Windows PowerShell、Windows コマンドプロンプト、または [conda](https://docs.conda.io/projects/conda/en/latest/user-guide/install/windows.html) からインストールして実行できます。
インストール後、Python ローカルパッケージの `Scripts` フォルダを `Path` 環境変数に手動で追加する必要がある場合があります。

`Scripts` フォルダのパスは次のようになります:

```bash
C:\Users\MyUserNameHere\AppData\Local\Packages\PythonSoftwareFoundation.Python.3.11_qbz5n2kfra8p0\LocalCache\local-packages\Python311\Scripts
```

システム上の `Scripts` フォルダー パスの `pip install` 出力メッセージを確認します。

## 最小限のPrefectインストール

`prefect-client` ライブラリは、Prefect Cloud またはリモートのセルフホスト型 Prefect サーバーインスタンスと連携するために設計された、Prefect の最小限のインストールです。

`prefect-client` は、Prefect の機能のサブセットを小さなインストールサイズで有効にするため、軽量環境、リソースが限られた環境、または一時的な環境での使用に最適です。
`prefect` ライブラリに含まれるすべての CLI およびサーバーコンポーネントは省略されています。

`prefect-client` の最新リリースをインストールするには、次のコマンドを実行します:

```bash
pip install -U prefect-client
```

## 次のステップ

APIサーバーも必要です。次のいずれかを選択してください。

* [Prefect Cloud](/v3/manage/cloud/)：強力なスケーリング、パフォーマンス、セキュリティを提供するマネージドソリューション。または
* [セルフホスト型Prefectサーバー](/v3/manage/server/)：お客様独自のインフラストラクチャ上で実行するAPIサーバー。スケーリング、認証、承認はすべてお客様ご自身で管理します。

Prefectのインストールが完了したら、[クイックスタート](/v3/get-started/quickstart/)に従ってお試しください。

リリースバージョンの詳細については、[Prefectリリースノート](https://github.com/PrefectHQ/prefect/releases)をご覧ください。

開発環境向けにPrefectをインストールする手順については、[貢献ドキュメント](/contribute/dev-contribute/)をご覧ください。
