---

この**自動化**セクションでは、自動的に実行され、イベントに応答するワークフローの作成方法について説明します。

- [フロー実行のスケジュール設定](/v3/automate/add-schedules/)では、特定の時間または間隔でフロー実行をスケジュールする方法について説明します。
- [イベントによるアクティビティの追跡](/v3/automate/events/events/)では、アクティビティの記録を作成するイベントを監視する方法について説明します。
- [イベントによるアクションのトリガー](/v3/automate/events/automations-triggers/)では、イベントを使用してアクションをトリガーする方法を示します。
- [カスタムイベントトリガーの定義](/v3/automate/events/custom-triggers/)では、高度なトリガーオプションについて説明します。
- [Webhookによるイベントの受信](/v3/automate/events/webhook-triggers/)では、Prefect CloudでWebhookを使用して外部システムからイベントを受信する方法について説明します。
- [サービス レベル アグリーメントによる信頼性の測定](/v3/automate/events/slas/)では、SLA を使用してデータ スタックの信頼性を測定する方法について説明します。
