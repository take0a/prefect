[Prefect CLI](/v3/get-started/install) は、Prefect サーバーのローカル インスタンスを起動する最も簡単な方法です。

### サーバーを起動する

1. ターミナルで「prefect server start」CLIコマンドを実行し、セルフホスト型PrefectサーバーインスタンスUIを起動します。

```bash
prefect server start
```

2. ブラウザでPrefectサーバーUIのURL（デフォルトでは[http://127.0.0.1:4200](http://127.0.0.1:4200)）を開きます。

![Viewing the dashboard in the Prefect UI.](/v3/img/ui/self-hosted-server-dashboard.png)

3. ターミナルで Ctrl + C を押して Prefect サーバーをシャットダウンします。

### サーバーの設定

ターミナルセッションに移動し、次のコマンドを実行して、API URL をセルフホスト型 Prefect サーバーインスタンスに設定するように設定します:

```bash
prefect config set PREFECT_API_URL="http://127.0.0.1:4200/api"
```


APIサーバーのアドレスはPrefectプロファイルに保存できます。
プロファイルがアクティブな場合、APIエンドポイントはそのアドレスに存在します。
プロファイルとPrefectの設定の詳細については、[プロファイルと設定](/v3/develop/settings-and-profiles/)を参照してください。
