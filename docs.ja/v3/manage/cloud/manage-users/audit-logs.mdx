---
title: クラウドアクティビティを監査する
description: Prefect Cloud の監査ログを使用して、ユーザー アクセスとアクティビティを監視します。
---

Prefect Cloud の [Pro プランと Enterprise プラン](https://www.prefect.io/pricing) では、監査ログを通じてコン​​プライアンスと透明性が強化されています。
監査ログは、アカウント内のメンバーが実行したアクティビティを時系列で記録し、セキュリティとコンプライアンスの観点から Prefect Cloud の詳細なアクションを監視できます。

監査ログは、Prefect Cloud アカウント内で誰が、いつ、どのようなアクションを実行し、どのリソースを使用したかを示します。

適切なツールと手順と組み合わせることで、監査ログは潜在的なセキュリティ違反の検出とアプリケーションエラーの調査に役立ちます。

監査ログには以下の情報が記録されます。

- ワークスペースへのアクセス
- ユーザーのログインアクティビティ
- ユーザーの API キーの作成と削除
- ワークスペースの作成と削除
- アカウントメンバーの招待と削除
- サービスアカウントの作成、API キーのローテーション、および削除
- セルフサービス価格帯の請求方法

## 監査ログの表示

アカウント設定ページから、**監査ログ** ページを選択すると、監査ログが表示されます。
管理者は以下の監査ログを表示できます。

- Prefect Cloud のアカウントレベルのイベント（例：
- メンバーの招待
- メンバーのロール変更
- Prefect Cloud へのメンバーのログインとログアウト
- サービスアカウントの作成または削除
- Prefect Cloud のワークスペースレベルのイベント（例：
- ワークスペースへのメンバーの追加
- メンバーのワークスペースロールの変更
- ワークスペースの作成または削除

管理者は、複数のディメンションで監査ログをフィルタリングし、ワークスペース、ユーザー、またはイベントの種類ごとに表示する結果を絞り込むことができます。
監査ログイベントは、**イベント** ドロップダウンメニューで確認できます。

監査ログは日付範囲でフィルタリングすることもできます。
監査ログの保存期間は [Prefect Cloud プラン](https://www.prefect.io/pricing) によって異なります。
