---
title: チームを管理する
description: Prefect Cloud でユーザーのチームを管理します。
---

Prefect Cloud の [エンタープライズプラン](https://www.prefect.io/pricing) では、アクセス制御ガバナンスを簡素化するチーム管理機能を提供しています。

アカウント管理者は、アカウント設定メニューから [チーム] をクリックし、チームとチームメンバーシップを設定できます。
チームは、ユーザーとサービスアカウントで構成されます。
チームは、ユーザーやサービスアカウントと同様に、ワークスペースまたはオブジェクトのアクセス制御リストに追加できます。

アカウントで SCIM が有効になっている場合、チームとその中のユーザーは IDP によって管理されます。
Prefect Cloud サービスアカウントは IDP によって管理されませんが、既存のチームに追加できます。

チームをサポートするためのオプションの詳細については、[Prefect Cloud プラン](https://www.prefect.io/pricing) をご覧ください。
