Prefect Cloud の [Pro および Enterprise プラン](https://www.prefect.io/pricing) では、特定のワークスペース内でチームメンバーのアクセスレベルを適切なレベルに設定できます。

ロールベースのアクセス制御 (RBAC) を使用すると、ユーザーに特定のアクティビティを実行するためのきめ細かな権限を割り当てることができます。

Enterprise アカウント管理者は、ユーザー向けにカスタムロールを作成し、Prefect に組み込まれているワークスペースロールの範囲を超える機能へのアクセスをユーザーに付与できます。

## 組み込みロール

ロールは、アカウントレベルまたは個々のワークスペースレベルでユーザーに権限を付与します。

- アカウントレベルのロールは、アカウント内でのユーザーのデフォルトの権限を定義します。
- ワークスペースレベルのロールは、特定のワークスペース内でのユーザーの権限を定義します。

以下のセクションでは、Prefect によって定義された組み込みのアクセス制御とワークスペースロールの機能について概説します。

| Viewer    | - ワークスペース内のフロー実行を表示します。<br /> - ワークスペース内のデプロイメントを表示します。<br /> - ワークスペース内のすべての作業プールを表示します。<br /> - ワークスペース内のすべてのブロックを表示します。<br /> - ワークスペース内のすべての自動化を表示します。<br /> - ワークスペースのハンドルと説明を表示します。 |

| Worker     | ワーカーが作業をポーリングして送信するために必要な最小限のスコープ。|

## カスタムワークスペースロール

組み込みロールはほとんどのユーザーのニーズに対応しますが、カスタムロールはユーザーにワークスペース内の特定の権限へのアクセス権限を付与します。

カスタムロールは組み込みロールから権限を継承できます。
これにより、チームのニーズに合わせてロールを微調整できると同時に、新機能が利用可能になった際にも、ユーザーはPrefectのデフォルトのワークスペースロール権限管理のメリットを享受できます。

{/*
<!-- vale off -->
*/}

Prefect の組み込みロールとは別に、カスタムワークスペースロールを作成できます。
このオプションにより、ワークスペース管理者はワークスペース機能へのユーザーアクセスを完全に制御できます。ただし、継承されていないカスタムロールの場合、新しい機能がリリースされるたびに、ワークスペース管理者が監視と権限設定の責任を負います。

{/*
<!-- vale on -->
*/}

カスタムロールに設定できる権限の詳細については、[ロールの権限](#workspace-role-permissions)をご覧ください。

新しいロールを作成すると、アカウントの**メンバー**ページと**ワークスペースの共有**ページで利用可能になり、ユーザーに適用できるようになります。

### 継承されたロール

カスタムロールを**継承されたロール**として設定できます。
継承されたロールを使用すると、組み込みのPrefectロールに関連付けられた初期権限セットからカスタムロールを作成できます。
カスタムロールには権限を追加できます。継承されたロールに含まれる権限は削除できません。

継承されたロールから作成されたカスタムロールは、新機能が利用可能になったときに、Prefectのデフォルトのワークスペースロール権限設定に従います。

カスタムロールと共に継承されたロールを設定するには、「**デフォルトロールから権限を継承する**」を選択し、新しいロールに権限を継承するロールを選択します。

## ワークスペースロールの権限

カスタムロールでは以下の権限が利用可能です。

### Automations

| Permission                           | Description                                                                                                     |
| ------------------------------------ | --------------------------------------------------------------------------------------------------------------- |
| View automations                     | ユーザーはワークスペース内で構成された自動化を確認できます。 |
| Create, edit, and delete automations | ユーザーはワークスペース内で自動化を作成、編集、削除できます。**自動化の表示**権限が含まれます。 |

### Blocks

| Permission                      | Description                                                                                                                          |
| ------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------ |
| View blocks                     | ユーザーはワークスペース内で構成されたブロックを表示できます。 |
| View secret block data          |ユーザーはワークスペース内で設定されたブロックとそのシークレットを確認できます。**ブロックの表示**権限が含まれます。 |
| Create, edit, and delete blocks | ユーザーはワークスペース内でブロックを作成、編集、削除できます。**ブロックの表示**と**シークレットブロックデータの表示**の権限が含まれます。 |

### Deployments

| Permission                  | Description                                                                                                                                                                                                                                                                                                                                                              |
| --------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| View deployments            | ユーザーはワークスペース内で構成されたデプロイメントを確認できます。  |
| Run deployments             | ユーザーはワークスペース内でデプロイメントを実行できます。ただし、デプロイメントに関連付けられたフローを実行する権限は付与されません。これは、ユーザー（キーを通じて）にデプロイメントを実行する権限のみを付与します。フローを実際に実行するには、適切なロールを持つサービスアカウントなど、別のユーザーまたはキーが必要です。**デプロイメントの表示** 権限が含まれます。 |
| Create and edit deployments | ユーザーはワークスペース内でデプロイメントを作成および編集できます。**デプロイメントの表示**および**デプロイメントの実行**の権限が含まれます。 |
| Delete deployments          |ユーザーはワークスペース内のデプロイメントを削除できます。**デプロイメントの表示**、**デプロイメントの実行**、**デプロイメントの作成と編集**の権限が含まれます。 |

### Flows

| Permission                                      | Description                                                                                                                                            |
| ----------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------ |
| View flows and flow runs                        | ユーザーはワークスペース内のフローとフロー実行を表示できます。 |
| Create, update, and delete saved search filters | ユーザーは、ワークスペース内で設定された保存済みのフロー実行検索フィルターを作成、更新、削除できます。**フローとフロー実行の表示**権限が含まれます。|
| Create, update, and run flows                   | ユーザーはワークスペース内でフローを作成、更新、実行できます。**フローとフロー実行の表示**権限が含まれます。 |
| Delete flows                                    | ユーザーはワークスペース内のフローを削除できます。**フローとフロー実行の表示** および **フローの作成、更新、実行** の権限が含まれます。 |

### Notifications

| Permission                            | Description                                                                                                                                                                |
| ------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| View notification policies            | ユーザーはワークスペース内で構成された通知ポリシーを確認できます。 |
| Create and edit notification policies | ユーザーはワークスペース内で設定された通知ポリシーを作成および編集できます。**通知ポリシーの表示**権限が含まれます。 |
| Delete notification policies          | ユーザーはワークスペース内で設定された通知ポリシーを削除できます。**通知ポリシーの表示**および**通知ポリシーの作成と編集**の権限が含まれます。 |

### Task run concurrency

| Permission                                  | Description                                                                                                                            |
| ------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------- |
| View concurrency limits                     | ユーザーは、ワークスペース内で構成されたタスク実行の同時実行制限を確認できます。 |
| Create, edit, and delete concurrency limits | ユーザーはワークスペース内でタスク実行の同時実行制限を作成、編集、削除できます。**同時実行制限の表示** 権限が含まれます。 |

### Work pools

| Permission                         | Description                                                                                                                                       |
| ---------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------- |
| View work pools                    | ユーザーはワークスペース内に構成された作業プールを表示できます。 |
| Create, edit, and pause work pools | User can create, edit, and pause work pools configured within a workspace. Includes permissions of **View work pools**.                           |
| Delete work pools                  | ユーザーはワークスペース内に構成されたワークプールを削除できます。**ワークプールの表示**および**ワークプールの作成、編集、一時停止**の権限が含まれます。 |

### Workspace management

| Permission                                        | Description                                                                                                       |
| ------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------- |
| View information about workspace service accounts | ユーザーはワークスペース内で構成されたサービス アカウントを表示できます。 |
| View information about workspace users            | ユーザーは、ワークスペースに招待されたユーザーのユーザー アカウントを表示できます。  |
| View workspace settings                           | ユーザーはワークスペース内で構成された設定を確認できます。  |
| Edit workspace settings                           | ユーザーはワークスペースの設定を編集できます。**ワークスペース設定の表示**権限が含まれます。 |
| Delete the workspace                              | ユーザーはワークスペースを削除できます。**ワークスペース設定の表示**および**ワークスペース設定の編集**の権限が含まれます。 |
