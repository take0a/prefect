[app.prefect.cloud](https://app.prefect.cloud) で Prefect Cloud アカウントにサインアップします。

<Note>
個々のユーザーを他のアカウントのメンバーになるよう招待することができます。
</Note>

## ユーザー設定

[プロフィールメニュー](https://app.prefect.cloud/my/profile)から、以下の個人設定にアクセスできます:

- **プロフィール**: 名前などの基本情報の表示と編集。
- **APIキー**: CLIやその他の環境からPrefect Cloudに接続するための[APIキー](/v3/manage/cloud/manage-users/api-keys/)の作成と表示。
- **設定**: カラーモードやデフォルトのタイムゾーンなどの設定を管理。
- **機能プレビュー**: 機能プレビューを有効または無効にできます。

## アカウントロール

アカウントに所属するユーザーは、管理者またはメンバーのロールを持つことができます。
管理者は、他のユーザーをアカウントに招待し、アカウントのワークスペースとチームを管理できます。

Pro および Enterprise レベルの Prefect Cloud アカウントの管理者は、アカウントのメンバーにワークスペース内の [ロール](/v3/manage/cloud/manage-users/manage-roles/)（ランナーやビューアーなど）を付与できます。
Enterprise レベルのアカウントでは、カスタムロールをご利用いただけます。

## APIキー

[APIキー](/v3/manage/cloud/manage-users/api-keys/)を使用すると、Prefect Cloud を使用する環境を認証できます。

## サービスアカウント (Pro) (Enterprise)

[サービスアカウント](/v3/manage/cloud/manage-users/service-accounts/) を使用すると、ユーザーアカウントに関連付けられていない Prefect Cloud API キーを作成できます。

## シングルサインオン（Pro）

エンタープライズプランでは、チームのIDプロバイダとの[シングルサインオン（SSO）](/v3/manage/cloud/manage-users/configure-sso/)連携が利用可能で、[ディレクトリ同期]および[SCIM Pro]のオプションもご利用いただけます。visioning](/v3/manage/cloud/manage-users/configure-sso/#directory-sync).

## 監査ログ (Pro) (Enterprise)

[監査ログ](/v3/manage/cloud/manage-users/audit-logs/) は、アカウントのメンバーである Prefect Cloud ユーザーが実行したアクティビティの時系列記録を提供します。

## オブジェクトレベルのアクセス制御リスト(Enterprise)

Prefect Cloud のエンタープライズプランでは、ワークスペース内の特定のユーザーとサービスアカウントへのアクセスを制限できるオブジェクトレベルのアクセス制御リスト（ACL）をご利用いただけます。

## チーム (Enterprise)

Prefect Cloud Enterprise アカウントのユーザーを [チーム](/v3/manage/cloud/manage-users/manage-teams/) に追加することで、アクセス制御ガバナンスを簡素化できます。
