[Terraform](https://www.terraform.io/) や [Helm](https://helm.sh/) などのツールを使用して、多くの Prefect リソースを管理できます。
これらのオプションは、Prefect の CLI や UI に代わる有効な手段となります。

## Terraform

このドキュメントは、Terraform でサポートされているすべての Prefect リソースについて説明しています。
この Terraform プロバイダーは Prefect チームによってメンテナンスされており、[Prefect API との同等性](https://github.com/PrefectHQ/terraform-provider-prefect/milestone/1) を実現するために積極的に開発が進められています。

Prefect チームは、[課題トラッカー](https://github.com/PrefectHQ/terraform-provider-prefect/issues) を通じて、貢献、機能リクエスト、バグレポートを歓迎します。

## Helm

各Helmチャートのサブディレクトリには、使用方法のドキュメントが含まれています。主なチャートは2つあります。

- `prefect-server`チャートは、Prefectサーバーをデプロイするために使用されます。これは、[Prefect Cloud](https://app.prefect.cloud/)の代替手段です。
- `prefect-worker`チャートは、[Prefectワーカー](/v3/deploy/infrastructure-concepts/workers)をデプロイするために使用されます。

最後に、`prefect-prometheus-exporter`チャートがあります。これは、監視とアラート用のPrefectメトリクスを公開するPrometheusエクスポーターをデプロイするために使用されます。
